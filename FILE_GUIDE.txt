========================================
TROUBLESHOOTING
========================================

If index.html doesn't work:
  → Make sure styles.css and script.js are in same folder
  → Try different browser (Chrome recommended)
  → Check browser console for errors

If PDF import fails:
  → Requires internet first time only
  → After that, works offline
  → Try re-importing or use .txt instead
