# Pro Teleprompter - Quick Start Guide

Welcome to Pro Teleprompter! This guide will get you up and running in minutes.

## Installation (Super Easy!)

1. **Extract the ZIP file** to any folder on your computer
2. **Double-click `index.html`** - it will open in your browser
3. **That's it!** No installation needed, works 100% offline

## First Time Setup (30 seconds)

### Step 1: Add Your Script
Choose one of these methods:
- **Type directly** in the text area
- **Click "Import"** to load a .txt, .md, or .pdf file
- **Copy & paste** your script from anywhere

### Step 2: Adjust Settings
- **Speed**: Use the slider (1-10) - start with 3 for normal pace
- **Font Size**: Adjust so you can read comfortably from your distance
- **Theme**: Choose Dark (default), Light, Green, or Blue

### Step 3: Start Reading!
- **Click "Play"** or press the **Spacebar**
- Text will smoothly scroll up
- Read from the center line (if enabled)

## Tips for Best Results

### Finding Your Perfect Speed
- **Slow (1-3)**: Technical content, unfamiliar material, slower speech
- **Medium (3-5)**: Normal conversation pace, most presentations
- **Fast (5-8)**: Energetic delivery, familiar content
- **Very Fast (8-10)**: Expert speakers, rapid delivery

### Font Size Recommendations
- **Close (2-3 feet)**: 36-48px
- **Medium (4-6 feet)**: 60-80px
- **Far (7-10 feet)**: 90-120px

### Theme Selection
- **Dark**: Best for low-light recording, easy on eyes (recommended)
- **Light**: Bright rooms, outdoor use
- **Green**: Green screen video production
- **Blue**: High contrast alternative
- **Custom**: Make it your own!
  - Click the "Custom" theme button
  - Use color pickers for Background, Text, and Accent colors
  - Try quick presets for inspiration
  - Perfect for matching brand colors or personal style

### Reading Guide Line
- The horizontal line shows where to focus your eyes
- Enable: Better tracking, maintains position
- Disable: Cleaner view, less distraction

Choose based on personal preference!

## Features Explained

### Import Files
**Supports**: .txt, .md, .pdf files

**To import**:
1. Click "Import" button
2. Select your file
3. Text appears automatically

**Note**: PDF import requires internet connection on first use only.

### Export Scripts
Save your script for backup or sharing:
1. Click "Export" button
2. File downloads automatically
3. Named with timestamp: `teleprompter-script-[timestamp].txt`

### Auto-Save
Your script and settings automatically save to your browser. When you reopen the teleprompter, everything is exactly as you left it!

**Important**: Auto-save doesn't work in private/incognito mode.

### Mirror Mode
Flips text horizontally for use with glass teleprompters. When reflected in the glass, text appears normal to you while camera looks through from behind.

## Troubleshooting

### Text won't scroll
- Make sure you have text entered
- Click Play or press Space
- Check that you haven't reached the end (press R to reset)

### Can't see controls
- Press **H** key to show/hide controls
- Press **Esc** to exit fullscreen

### PDF shows weird symbols
- Make sure you have internet connection (first time only)
- Try re-importing the PDF
- If still not working, convert PDF to text first

### Speed is too fast/slow
- Use speed slider or **â†‘**/**â†“** arrow keys
- Start with speed 3 and adjust from there

### Script doesn't save
- Don't use private/incognito browser mode
- Check browser settings allow localStorage
- Try exporting script as backup

### Choppy scrolling
- Close other browser tabs
- Use Chrome or Edge for best performance
- Lower font size slightly
